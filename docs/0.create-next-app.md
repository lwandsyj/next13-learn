1. create-next-app 是快速创建next.js 应用的脚手架

    语法：

        npx create-next-app < projectName >

    参数:

    + -V: 当前版本号

    + --ts: 使用typescript

    + --js: 使用js

    + --eslint: 添加eslint

    +
